# CS-499 Capstone Project – Cody Leveille

This repository contains the completed Computer Science Capstone project for **CS-499** at **Southern New Hampshire University**.  
It serves as a professional ePortfolio that demonstrates my growth, technical skills, and readiness to contribute in a professional software development environment.

The project includes an original artifact, three targeted enhancements, three enhancement naratives, a formal code review, and a professional self-assessment aligned with CS-499 course outcomes.
